# Customer Behavior & Product Usage Analysis

## Overview
This project analyzes customer usage data to understand engagement patterns,
identify drop-off trends, and generate insights to improve product experience.

## Objective
- Explore customer behavior using data analysis
- Formulate and validate hypotheses
- Provide product-focused insights

## Tools
- Python
- Pandas, NumPy
- Matplotlib
- Jupyter Notebook

## Key Steps
1. Data cleaning and validation
2. Exploratory analysis (EDA)
3. Hypothesis testing
4. Insight generation

## Key Insights
- Users with higher session duration stay engaged longer
- Feature usage patterns highlight retention opportunities
- Early drop-off indicates onboarding improvement points

## Limitations
- Dataset lacks demographic detail
- Future work can include predictive modeling
